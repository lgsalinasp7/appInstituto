⚡ Resumen: el viaje completo
💻
Dirección IP — Tu computadora
Tu PC tiene una IP privada (ej. 192.168.1.5) asignada por el router via DHCP. El router tiene una IP pública que es la visible en internet.
~0ms
🌐
DNS — Traducción de nombre a IP
kaledsoft.tech → 93.184.216.34. Tu OS consulta la jerarquía DNS: cache → resolver → root → TLD → autoritativo.
~10ms
🤝
TCP — Conexión confiable
3-way handshake (SYN → SYN-ACK → ACK). Garantiza que los paquetes lleguen en orden y completos.
~80ms
🔐
TLS — Cifrado del canal
El candado 🔒. Negocia claves de cifrado. Nadie en el camino puede leer tus datos.
~120ms
📡
HTTP/2 — La petición
GET /dashboard con headers. El servidor responde 200 OK + HTML. Multiplexing permite múltiples assets en paralelo.
~200ms
🌍
CDN — Servidor más cercano
Cloudflare/Vercel Edge sirve el contenido desde el nodo más cercano a ti. Sin CDN serías 150ms+ más lento.
~12ms
🖥️
Browser — Render final
HTML → DOM, CSS → CSSOM, JS ejecutado. Paint y composite. Ves la página. Total: ~380ms desde el primer keystroke.
~380ms
