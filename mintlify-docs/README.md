# Mintlify Docs - App Instituto

## Ejecutar localmente

1. Instala la CLI de Mintlify:

```bash
npm i -g mint
```

2. Entra a esta carpeta:

```bash
cd mintlify-docs
```

3. Inicia el entorno local:

```bash
mint dev
```

La documentacion se abrira en `http://localhost:3000`.

## Publicar

- Sube estos archivos al repositorio conectado en Mintlify.
- Mintlify desplegara automaticamente los cambios.

## MCP de esta documentacion

Cuando este sitio este publicado, conecta su MCP en Cursor usando:

`https://<tu-dominio-docs>/mcp`
