# Testing

## Suite disponible

- Unit/integration: Vitest
- Linting: ESLint

## Comandos

```bash
npm run lint
npm run test:run
npm run test:coverage
```

## Flujo recomendado antes de merge

1. Ejecutar `npm run lint`.
2. Ejecutar `npm run test:run`.
3. Verificar que el build no rompa:

```bash
npm run build
```
